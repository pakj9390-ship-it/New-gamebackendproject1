# Context
You are working on a game backend admin management system built with:
- **Backend**: Node.js + Express + SQLite3
- **Frontend**: React 18 + Vite + Tailwind CSS
- **Auth**: JWT token stored in localStorage

The project structure:
```
server/
  server.js      # Express API routes
  db.js          # SQLite database setup
cient/src/
  api.js         # API client functions
  components/    # React components
```

# Task
Implement a complete feature module for the admin dashboard:

1. **Database Schema** - Add new table(s) or columns to `server/db.js`
2. **Backend API** - Add REST endpoints to `server/server.js`:
   - `GET /api/{resource}` - List with pagination
   - `POST /api/{resource}` - Create
   - `PUT /api/{resource}/:id` - Update
   - `DELETE /api/{resource}/:id` - Delete
3. **Frontend API** - Add functions to `client/src/api.js`
4. **UI Components** - Create React components:
   - Table view with pagination
   - Create/Edit modal form
   - Search functionality
5. **Navigation** - Add tab/button to switch to the new view

# Rules
- Use existing code patterns (JWT auth, SQLite promisified, Tailwind styling)
- Keep components in `client/src/components/`
- Use the existing `api.js` pattern for HTTP requests
- Support pagination (page/limit/search query params)
- Use Chinese UI labels where appropriate
- Add test data via API calls for verification

# Output Format
Provide the complete code for all modified/created files. Use edit tools to apply changes.
